RoadFlow gives you the best of both worlds: visual drag-and-drop for quick automations, and full code nodes for complex logic. Write JavaScript, Python, or shell scripts directly in the flow editor.

  • Visual canvas with snap-to-grid alignment
  • JavaScript and Python code nodes
  • Conditional branching with boolean logic
  • Loop and iterator nodes for batch processing
  • Error handling with retry and fallback paths
  • Sub-workflow composition for reusable logic
// RoadFlow — custom transform node
export default async function transform(input) {
  // Access upstream node data
  const records = input.getAll('webhook');

  // Call local Ollama model
  const enriched = await Promise.all(
    records.map(async (record) => {
      const summary = await ollama({
        model: 'llama3',
        prompt: `Summarize: ${record.text}`,
      });
      return { ...record, summary };
    })
  );

  // Pass to next node
  return enriched.map(item => ({
    json: item,
    binary: {},
  }));
}

Built for Real Work

RoadFlow isn't a toy. It's production infrastructure for automating complex business processes, data pipelines, and AI workflows.

🔌

200+ Built-in Nodes

HTTP, databases (PostgreSQL, SQLite, Redis), file systems, email, Slack, Discord, Git operations, SSH commands, S3/MinIO, and more. Every node is fully configurable with expression support for dynamic values.

🤖

AI-Native Nodes

Connect to local Ollama models, OpenAI, Anthropic, or any OpenAI-compatible endpoint. Embeddings, summarization, classification, extraction — all as drag-and-drop nodes. Chain multiple AI calls with context passing.

🕑

Cron & Webhook Triggers

Schedule workflows with cron expressions or trigger them via webhooks, file watchers, email arrival, database changes, or MQTT messages. Every trigger supports filtering and deduplication.

🛠

Error Handling & Retry

Built-in retry logic with exponential backoff. Dead letter queues for failed items. Error branches let you define fallback behavior. Automatic alerting on workflow failures via any notification channel.

📊

Execution History & Logs

Every workflow execution is logged with full input/output data at each node. Replay failed executions. Debug with step-through mode. Export execution data for compliance and auditing.

🌐

Pi Fleet Integration

Native support for BlackRoad Pi fleet services. SSH into any Pi, run commands, collect sensor data, deploy services. RoadFlow treats your entire home lab as a first-class automation target.
